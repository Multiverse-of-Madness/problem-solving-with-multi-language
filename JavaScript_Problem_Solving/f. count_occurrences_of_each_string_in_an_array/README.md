# Question: Count occurrences of each string in an array.

## Input:

    const array = ['green', 'red', 'blue', 'red', 'yellow', 'yellow', 'green', 'red', 'blue', 'green', 'yellow', 'red'];


## Output:

    { green: 3, red: 4, blue: 2, yellow: 3 }



## This problem is solve two types:

1. Use for loop.

2. Use reduce() method.
